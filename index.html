Blackpink fan website CI/CD

    저는 블랙핑크의 팬 페이지를 만들고싶었고, 위 사진은 그 설계도 입니다.
    매일매일 블랙핑크 맴버들의 인스타그램 사진을 크롤링해서 S3 버킷에 이미지를, MySQL 서버에 메타데이터를 저장한 후, 유저들이 그 사진을 다운로드 받을 수 있게끔 만들고 싶었습니다.

    하지만 동작하는 앱을 구현하던 중 다른 블랙핑크 팬 앱, 웹 들에 비해 퀄리티가 떨어진다고 생각해서 실제 앱 배포와 운영은 하지 않고 중단했습니다.

가상화폐 서버리스 자동매매

    조코딩 유튜브 영상의 프로젝트를 서버리스로 업그레이드한 가상화폐 자동매매 프로젝트 입니다.
    여러 다른 회사들의 open api 에 관심을 가지고 찾아보던 중, 증권사나 거래소 api 를 활용해서 자동매매를 하는 것에 관심이 생겨 시작하게 됬습니다.
    처음에는 단순 목표값만을 설정한 파이썬 스크립트 하나만 EC2 에서 돌아갔었지만,
    이후 페이스북의 prophet 라이브러리를 사용해 종가를 예측해주는 워크로드를 위해 Serverless 로 전체 로직을 마이그레이션 했습니다.
    현재는 리소스들을 개별 배포+운영 하고있지만 테라폼이나 Cloudformation 혹은 AWS-CDK python 으로 관리하는 로직을 추가할 예정입니다.
