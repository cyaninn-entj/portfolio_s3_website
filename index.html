구름 쿠버네티스 전문가 양성 과정(6기)

    2022년 7월에 구름 K-Digital Traning 쿠버네티스 양성 과정(6기) 에 참가했었습니다.
    Ansible, docker 등을 다뤘었고, 마이크로서비스 운영을 위한 쿠버네티스에 대한 이해와 애플리케이션 운영 등을 배울 수 있었습니다.
    AWS 클라우드 안에서 k8s 클러스터를 직접 구성하여 운영했었고, EKS 로 마이그레이션 한 이후 그 강력한 기능이 기억에 남습니다.
    교육 과정 마무리엔 4주간의 짧은 프로젝트를 통해 k8s 운영시 일어날 수 있는 트러블슈팅을 해볼 수 있어서 좋았고, CI/CD 구축을 구현해 볼 수 있어 좋은 경험이 되었습니다. 마이크로서비스와 DevOps 워크로드 오케스트레이션 플랫폼인 쿠버네티스를 공부하는데 있어서 좋은 이정표가 된 교육과정이였습니다.

개인 환경 인프라 CI/CD

    저는 블랙핑크의 팬 페이지를 만들고싶었고, 위 사진은 그 설계도 입니다.
    매일매일 블랙핑크 맴버들의 인스타그램 사진을 크롤링해서 S3 버킷에 이미지를, MySQL 서버에 메타데이터를 저장한 후, 유저들이 그 사진을 다운로드 받을 수 있게끔 만들고 싶었습니다.
    개발환경에서 테스트 후 운영환경에서의 빠른 적용을 위해서 Jenkins server 용 인스턴스를 만들고 github repository 와 hub.docker repository 를 사용해 CI/CD Pipeline 을 구성해봤었습니다.
    하지만 동작하는 앱을 구현하던 중 다른 블랙핑크 팬 앱, 웹 들에 비해 퀄리티가 떨어진다고 생각해서 실제 앱 배포와 운영은 하지 않고 중단했습니다.
